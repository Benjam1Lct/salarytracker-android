Suppression des données

Application : SalaryTracker

Vous pouvez supprimer définitivement votre compte et l'intégralité de vos données à tout moment. Deux méthodes sont possibles.

Méthode 1 — Directement dans l'application (recommandé, immédiat)

  1. Ouvrez SalaryTracker et connectez-vous.
  2. Allez dans Paramètres (icône en haut de l'écran).
  3. Dans la section Profil & Compte, appuyez sur « Supprimer mon compte ».
  4. Confirmez. Vos données sont effacées immédiatement et vous êtes redirigé vers l'écran de connexion.

Méthode 2 — Par e-mail

Si vous n'avez plus accès à l'application, envoyez une demande à benjaminlecomte37@gmail.com en indiquant l'adresse e-mail ou le numéro de téléphone associé à votre compte. La suppression sera effectuée sous 30 jours.

Données supprimées

La suppression efface définitivement :

  • votre compte d'authentification (Google / Facebook / téléphone) ;
  • tous vos emplois, contrats, journées de travail et horaires types ;
  • vos règles de saisie automatique et bulletins de paie importés ;
  • votre éventuelle clé API Gemini enregistrée.
⚠️ La suppression est irréversible. Une fois effectuée, les données ne peuvent pas être récupérées.
